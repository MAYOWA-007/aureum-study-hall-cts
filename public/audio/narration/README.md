# Aureum CTS narration pack

This public directory contains only generated narration assets and public metadata. It must never contain a Fish Audio API key or another private credential.

`manifest.json` is generated from the complete CTS question bank. The approved pack uses Fish Audio S2.1 Pro and voice reference `f3b85abdea0e4e1c9fc60b4e3063a4a0`; `pack.json` remains in `awaiting_generation` status until its static audio files are produced by the private generation command.
